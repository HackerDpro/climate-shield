Yes, you do have to update it manually here—but don’t worry, you can easily automate it later using what is called a **Git Webhook** (which replicates exactly what Render does).

By default, Nest won't know when you push code to GitHub. Here is how you do it manually, followed by how to make it automatic.



































































































































































































Whenever you push a new change from your Codespace to GitHub, you just need to log into your Nest server and run a quick 3-step sequence:

1. **Log in:**
```bash
ssh diego@hackclub.app

```


2. **Pull your new code:**
```bash
cd your-repo-name/backend
git pull

```


3. **Tell PM2 to reload your app:**
```bash
pm2 restart fastapi-backend

```



*Note: If you added new packages to `requirements.txt`, you will also need to run `pip install -r requirements.txt` before restarting PM2.*
